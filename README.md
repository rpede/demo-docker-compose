# CDS 2024 Proof-of-concept

## Quick-start

```sh
docker compose up
```

## Services

| Sub-system  | URL                                                    |

| MailCatcher | <http://localhost:1080/>                               |
| Postgres    | <postgres://postgres:mysecret@localhost:5432/postgres> |
